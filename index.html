A brief intro to python

Introduction

Python is a cross-platform programming language, which means that it can run on multiple platforms like Windows, macOS, Linux, and has even been ported to the Java and .NET virtual machines. It is free and open-source.

Even though most of today's Linux and Mac have Python pre-installed in it, the version might be out-of-date. So, it is always a good idea to install the most current version.

Python Setup

There are so many ways to run Python. If you want to code Python from your browser then my suggestion would be Replit.

But if you want to do Python development locally, then you will have to install a few things in order to setup your local machine for coding.

Follow the following steps to run Python on your computer:

  • Download Python and Visual Studio Code
  • Install Python and Visual Studio Code
  • Open Visual Studio Code
  • Open a new file and save it with .py extension
  • Write your code and run it.
First Code

Let's start with a classic "Hello, World!" program. It's the customary way to greet the programming world!

print("Hello, World!")

Save this code in a file with a .py extension (e.g., hello.py). Open your terminal, navigate to the file's directory, and run the program using the command python hello.py. You should see the output "Hello, World!" displayed on your screen.

Variables

Variables in Python are used to store data values. You don't need to explicitly declare the variable's type; Python infers it from the assigned value. Here's an example:

name = "John"
age = 24
gamer = False
Data Types

In computer programming, data types specify the type of data that can be stored inside a variable. Python supports various data types. Some common ones are:

  • Integers: whole numbers (e.g., 42, -17)
  • Floats: numbers with decimal points (e.g., 3.14, -0.5)
  • Strings: sequences of characters (e.g., "Hello", 'Python')
  • Booleans: represent truth values (True or False)
# Examples of data types
age = 25
pi = 3.14
name = "Alice"
is_student = True

print(age, pi, name, is_student)
Operators

Arithmetic operators are used to perform mathematical operations like addition, subtraction, multiplication, etc. Operators are symbols that perform operations on variables and values. Python supports various types of operators:

  • Arithmetic operators: +, -, *, /, // (floor division), % (modulo)
  • Comparison operators: ==, !=, >, <, >=, <=
  • Logical operators: and, or, not
a = 7
b = 2

# addition
print ('Sum: ', a + b)  

# subtraction
print ('Subtraction: ', a - b)   

# multiplication
print ('Multiplication: ', a * b)  

# division
print ('Division: ', a / b) 

# floor division
print ('Floor Division: ', a // b)

# modulo
print ('Modulo: ', a % b)  

# a to the power b
print ('Power: ', a ** b)
Conditional Statements

Conditional statements allow you to make decisions in your code based on certain conditions. Python uses the if, elif, and else keywords for this purpose.

# Example of a conditional statement
num = int(input("Enter a number: "))

if num > 0:
    print("The number is positive.")
elif num < 0:
    print("The number is negative.")
else:
    print("The number is zero.")
Loops

In computer programming, loops are used to repeat a block of code. Loops enable you to execute a block of code repeatedly.

For example, if we want to show a message 100 times, then we can use a loop. It's just a simple example; you can achieve much more with loops.

There are 2 types of loops in Python:

  • for loop
  • while loop
Functions

Functions are blocks of reusable code that perform specific tasks. They help in organizing your code and making it more manageable. A block of code that performs a specific task.

# Example of a function
def greet(name):
    print("Hello, " + name + "! Welcome to Python programming.")

# Function call
greet("John")
